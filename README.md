This project explores the Lonely Runner Conjecture through a computational lens, focusing on the practical verification of cases involving 11 to 13 runners.

The conjecture considers runners moving at distinct constant speeds on a unit circle and predicts that each runner will eventually be isolated from all others by a fixed minimum distance. While simple to state, the problem becomes increasingly complex as the number of runners grows, making direct analytical approaches difficult.

Instead of pursuing a purely theoretical proof, this work follows a computer-assisted strategy. By reducing the problem to a finite set of cases and leveraging existing public implementations, the project aims to examine how far current computational methods can verify the conjecture in practice.

The goal is not only to study the mathematical structure of the problem, but also to investigate reproducibility — what can actually be confirmed on a local machine versus what is claimed in recent research.
